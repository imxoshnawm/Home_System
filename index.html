Control of Balcony Lighting

Controlling the light on the balcony with contact light, automatically turns off when the light is exposed and turns on in the dark.

    const int GlopPin = 8;  
    const int ldrPin = A0;  // the number of the LDR pin
                                
    void setup() {
      Serial.begin(9600);
      pinMode(GlopPin, OUTPUT);  // initialize the LED pin as an output
      pinMode(ldrPin, INPUT);   // initialize the LDR pin as an input
    }
                                
    void loop() {
      int ldrStatus = analogRead(ldrPin);  // read the status of the LDR value

Smart Garage

Is an automated system that uses sensors and connected technology to perform specific actions based on environmental conditions. For example, a smart garage can detect rain using a rain sensor and automatically open a roof to provide shelter for a car, preventing it from getting wet.

Diagram of project

Project Code

#include 

Servo myservo;
int rainSensorPin = 2;  // Water sensor signal pin
int servoPin = 9;       // Servo control pin
bool waterDetected = false;  // Track water detection state

void setup() {
  Serial.begin(9600);        // Initialize serial monitor for debugging
  
  pinMode(rainSensorPin, INPUT);  // Set sensor pin as input
  
  myservo.attach(servoPin);  // Attach servo to pin
  myservo.write(0);         // Initialize at 0 degrees
}

void loop() {
  waterDetected = digitalRead(rainSensorPin);  // Read sensor state
  
  if (waterDetected) {
    Serial.println("Water detected! Servo at 90 degrees.");
    myservo.write(90);  // Open roof to 90 degrees
  } else {
    Serial.println("No water detected. Servo at 0 degrees.");
    myservo.write(0);   // Return to 0 degrees
  }
  
  delay(1000);  // Short delay to keep loop running smoothly
}

Lighting Control

The system allows for control of room lighting via voice commands that turn the lights on and off with clap recognition.

Diagram of project

Project Code

int soundSensor = 2; // Connect sound sensor to digital pin 2
int LED = 13; // Onboard LED
int soundState = 0;

void setup() {
  pinMode(soundSensor, INPUT); // Setting the sound sensor as input
  pinMode(LED, OUTPUT); // Setting the pin 13 as output for turning on/off LED
  Serial.begin(9600);
}

void loop() {
  soundState = digitalRead(soundSensor); // Reading the sound sensor status
  
  if (soundState == 1) { // If the sound sensor detects sound
    if (digitalRead(LED) == LOW) {
      digitalWrite(LED, HIGH); // Turn on LED
      Serial.println("LED On");
    } 
    else {
      digitalWrite(LED, LOW); // Turn off LED
      Serial.println("LED Off");
    }
    delay(500); // To avoid multiple triggers from one clap
  }
}
